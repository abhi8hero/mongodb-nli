# MongoDB NLI

A natural language interface (NLI) for MongoDB that allows users to interact with their database using simple English commands.


## 📖 About the Project

**MongoDB NLI** is a web-based application designed to simplify database operations by converting natural language queries into MongoDB commands.  
The project focuses on practical JavaScript implementation, MongoDB integration, and real-world database management workflows.

This project was built to enhance understanding of:
- Natural language processing
- Database querying
- Frontend-backend integration
- Real-world MongoDB operations

## ✨ Features
- Convert natural language queries into MongoDB commands
- Real-time query execution
- Display query results dynamically
- Clean and user-friendly UI
- Simple authentication & session management
- Editable AI Generated Query, JSON Format and also Table View
- Specialy build for non technical employees/person


## 🛠️ Tech Stack
- **React** – Frontend logic
- **Node.js** – Backend API
- **MongoDB** – Database operations
- **Natural Language Processing (NLP)** – Query parsing
- **Google Gemini** – AI Generated Response
- **Fast API** – Middle level query handling API
